### Description
This is a standard starter with Typescript, TSLint, Prettier, Lint-Staged(Husky) and Sass

### Keywords
    - TypeScript
    - TSLint
    - Prettier
    - Lint-Staged
    - Styling:SCSS

### Features
    - Category and Tag for post
    - Type Safe by TypeScript
    - Format Safe by TSLint and Prettier with Lint-Staged(Husky)
